# Machine-Readable Schemas

JSON Schema definitions for gh-actions-doctor data formats.

| Schema | Description |
| --- | --- |
| [`json-output.schema.json`](json-output.schema.json) | Scan result JSON output format. |
| [`baseline.schema.json`](baseline.schema.json) | Baseline suppression file format. |
| [`configuration.schema.json`](configuration.schema.json) | `.gh-actions-doctor.yml` configuration file format. |

These schemas follow JSON Schema draft 2020-12.
